## menu

Проект представляет собой REST API для управления меню ресторана. 
Реализована иерархическая структура: меню → подменю → блюда. 
Каждый уровень содержит CRUD операции. API документировано с помощью Swagger 
(доступно по адресу /docs).

### Технологии

- **FastAPI** — веб-фреймворк для создания API
- **PostgreSQL** — реляционная база данных
- **SQLAlchemy** — ORM для работы с базой данных
- **Databases** — асинхронная поддержка SQLAlchemy
- **pydantic** — валидация данных
- **Docker** — контейнеризация приложения и базы данных
- **Docker Compose** — оркестрация многоконтейнерного приложения
- **Uvicorn** — ASGI-сервер для запуска FastAPI

### Настройка проекта
Создайте `.env` файл в корне репозитория:
```bash
cp .env.dist .env
```

### Сборка образов и запуск контейнеров
В корне репозитория выполните команду:
```bash
docker-compose up --build
```

### Остановка контейнеров
Для остановки контейнеров выполните команду:
```bash
docker-compose stop
```

Проект доступен по адресу http://127.0.0.1:8000/docs
